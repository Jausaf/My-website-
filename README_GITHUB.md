# Hotel Buddha Residency & Family Restaurant - Website

A premium, modern website for Hotel Buddha Residency & Family Restaurant located in Kiul, Lakhisarai, Bihar.

## 📋 Project Overview

This is a fully responsive, production-ready website built with React, TypeScript, and Tailwind CSS. The site showcases hotel accommodations, dining services, and provides a seamless user experience across all devices.

**Live Website:** [hotelresidency-a3vyfccy.manus.space](https://hotelresidency-a3vyfccy.manus.space)

## ✨ Features

### Pages
- **Home** - Hero section with key statistics and amenities showcase
- **Menu** - Complete dining menu (28 items) with professional food photography and pricing
- **About Us** - Hotel story, amenities, and testimonials
- **Visit Us** - Location map, directions, and contact information

### Key Functionality
✓ Responsive design (mobile, tablet, desktop)

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- pnpm (or npm)

### Installation

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Preview production build
pnpm preview
```

## 🔧 Customization

### Update Menu Items
Edit `client/src/data/menu.ts` to add, remove, or modify menu items.

### Change Colors
Update CSS variables in `client/src/index.css` to customize the color scheme.

### Update Content
Edit individual page files in `client/src/pages/` to update page content.

### Add New Pages
Create new page components in `client/src/pages/` and add routes in `client/src/App.tsx`.
